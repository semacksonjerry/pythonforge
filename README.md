## 🛠️ Requirements

Before starting, ensure you have:

- Python 3.10+
- Git installed
- Internet connection
- A code editor (VS Code recommended)

---

## ⚡ Step 1: Install UV (Environment Manager)

We use **UV** to manage Python environments automatically.

**Install UV — Mac/Linux:**

```bash
curl -Ls https://astral.sh/uv/install.sh | bash
```

**Install UV — Windows (PowerShell):**

```powershell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

Restart your terminal after installation.

**Verify (all platforms):**

```bash
uv --version
```

---

## 📥 Step 2: Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/pythonforge.git
cd pythonforge
```

---

## 📦 Step 3: Setup Project Environment

Inside the project folder, run:

```bash
uv sync
```

This will:

- Create a virtual environment (`.venv`)
- Install all required dependencies
- Lock package versions for consistency

---

## 📚 Step 4: Run Jupyter Lab (Start Learning)

Launch the lab environment:

```bash
uv run jupyter notebook
```

OR (recommended):

```bash
uv run jupyter lab
```
